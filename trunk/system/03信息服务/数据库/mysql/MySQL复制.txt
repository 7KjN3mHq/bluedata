在 master 上：
mysql> GRANT REPLICATION SLAVE ON *.* TO 'slave'@'%' IDENTIFIED BY 'slave';

修改配置文件：
## master
server-id        = 170
log-bin          = /home/log-bin/mysql-bin
binlog-do-db     = test
binlog-ignore-db = mysql information_schema

## slave
server-id        = 162
master-host      = 192.168.0.170
master-port      = 3307
master-user      = slave
master-password  = slave
relay-log        = /home/log-bin/relay-bin
replicate-do-db  = test
binlog-ignore-db = mysql information_schema

查看信息：
SHOW MASTER STATUS;
SHOW SLAVE STATUS;
