ssh 的 key 验证：

# ssh-keygen -t rsa

直接回车

# ssh SERVER

# mkdir .ssh

# chmod 0600 .ssh

# exit

# scp .ssh/id_rsa.pub root@SERVER:.ssh/authorized_keys

增加其它公钥：
# cat /home/id_rsa.pub >> authorized_keys

puttygen 生成的公钥文件转换为 OpenSSH 程序使用的格式：
ssh-keygen -i -f /root/.ssh/id_rsa.pub > /root/.ssh/authorized_keys


chmod -R 600 /root/.ssh
